🚀 HOW TO RUN (Choose One Method)
═══════════════════════════════════════════════════════════════

METHOD 1: AUTOMATIC STARTUP (Recommended)
──────────────────────────────────────────────────────────────

Windows Users:
  1. Double-click "run.bat"
  2. Automatic installation will start
  3. Browser opens to http://localhost:5000

Mac/Linux Users:
  1. Open Terminal
  2. Navigate to this folder
  3. Type: bash run.sh
  4. Browser opens to http://localhost:5000


METHOD 2: MANUAL SETUP
──────────────────────────────────────────────────────────────

Step 1: Install Python 3.7+
  Download from: https://www.python.org/downloads/

Step 2: Install Dependencies
  pip install -r requirements.txt

Step 3: Run the App
  python app.py

Step 4: Open Browser
  Go to: http://localhost:5000


📝 HOW TO USE THE APPLICATION
═══════════════════════════════════════════════════════════════

1. ENTER SCHOOL DETAILS
   ├─ School Name (required)
   ├─ Inspection Date (required)
   ├─ Inspector Name
   └─ District/Region

2. COMPLETE INSPECTION
   ├─ Click category to expand
   ├─ Rate each item (1-5 scale)
   ├─ Add notes if needed
   └─ Repeat for all 6 categories

3. VIEW RESULTS
   ├─ Overall score updates automatically
   ├─ See status (Good/Fair/Poor)
   └─ Review category breakdown

4. DOWNLOAD REPORT
   ├─ Click "Download PDF" for official report
   ├─ Or reset and start new inspection
   └─ PDF saves to your Downloads folder


📊 6 INSPECTION CATEGORIES
═══════════════════════════════════════════════════════════════

1. 🏢 Infrastructure & Facilities
   - Building structure, classrooms, labs, library, sports

2. 🧹 Cleanliness & Hygiene
   - Classrooms, restrooms, water, waste, pest control

3. 🛡️ Safety & Security
   - Fire safety, exits, first aid, security, CCTV

4. 📚 Academic Standards
   - Curriculum, teachers, materials, performance, assessment

5. 👥 Staff & Management
   - Qualifications, counselor, development, administration

6. ❤️ Student Welfare
   - Health services, mental health, anti-bullying, activities

═══════════════════════════════════════════════════════════════

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Port 5000 already in use
Solution: Edit app.py and change port to 5001 or 8000

Problem: Python not found
Solution: Install Python 3.7+ and add to PATH

Problem: ModuleNotFoundError
Solution: Run: pip install -r requirements.txt

Problem: Port connection refused
Solution: Make sure Flask is running and try http://localhost:5000
